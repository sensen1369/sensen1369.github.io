使用hexo+github搭建个人博客

发表于 2022-06-10

安装Node.js

地址：http://nodejs.cn/download/

安装Git

地址：https://git-scm.com/downloads

验证安装是否成功

使用 Win+R cmd 打开命令行

会显示各个版本，如图：

安装hexo

验证

结果如图：

初始化hexo

在想要存放博客文件的目录下，右键，选择git bash here

blog为创建的文件夹名

安装完成后会生成以下内容，主要为：

文件名 描述
node_modules 依赖包
scaffolds 生成文章的一些模版
source 你的文章目录
themes 主题
_config.yml 配置文件

根据md生成页面，会生成一个public的文件夹

打开本地服务
浏览器打开http://localhost:4000即可看到生成页面，会比较慢

页面显示如下即成功：

Ctrl+C关闭服务

创建github仓库

hexo部署

打开blog文件夹下的_config.yml文件，拉到最后，修改

name是创建的仓库名字

安装deploy-git

是部署命令，可以将文件上传到github仓库

清除之前生成的页面，即删除public文件夹

生成页面，部署页面

完成

浏览器打开https://name.git.io即可看到页面

其他设置

hexo配置文件修改

博客根目录下的_config.yml是整个hexo框架的配置文件，详细可参考官方的配置描述。

Site

名称 描述
title 网站标题
subtitle 网站副标题
description 网站描述
keywords 关键词
author 作者
language 网站语言
timezone 时区

修改建议：

注意：
配置’ : ‘后要加空格

URL

名称 描述
url 网站
permalink 网站链接格式

修改建议：

参数 结果
:year/:month/:day/:title/ 2020/06/10/hello-world/
:year/:month-:day-:title/ 2020/06-10-hello-world/

Extensions

下面就有个themes，配置页面主题，默认landscape主题。

更改主题

官网主题地址：https://hexo.io/themes/
或者去github搜索hexo theme都能找到一大推主题

以hexo-theme-fluid为例
项目地址为：https://github.com/fluid-dev/hexo-theme-fluid

项目下方都会写明使用方法

以下简单介绍一下：

进入网站根目录blog，右键git bash here

修改博客配置文件_config.yml，注意是根目录下的，不是主题目录下的
